Instrucciones:

1) Ejecutar por consola ```git clone git@github.com:javierITB/E-Ciclos.git```
2) Una vez adentro de la carpeta raiz ```E-Ciclos```, se debe ejecutar el siguiente comando de docker:

```
docker compose up
```

Cabe a destacar que la ejecución de docker se demorará 3 minutos aproximadamente debido al tamaño del dataset.

3) abrir el enlace: http://127.0.0.1:8050/
4) Escribir la dirección de origen (Ejemplo: Los Dominicos) y luego presinar el boton de fijar
5) Escribir la dirección de destino (Ejemplo: Plaza Maipu) y luego presinar el boton de fijar
6) Presionar el boton para calcular ruta


Para seleccionar otros nodos se debe presionar en el boton Resetear selección.
